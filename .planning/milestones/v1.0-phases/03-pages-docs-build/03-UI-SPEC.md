## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0F0F11` (`--gray-0`) | Page background, navbar background, mobile nav overlay |
| Secondary (30%) | `#16161A` (`--gray-1`) | Code block backgrounds, card/block surfaces |
| Accent (10%) | `#3A5ECF` (`--brand-color`) | See reserved list below |
| Destructive | none | No destructive actions in this phase |

Accent (`--brand-color: #3A5ECF`) reserved for:
- Primary CTA button background (`brand` theme on `Button`)
- Text selection highlight (`color-mix(in srgb, var(--brand-color) 60%, transparent)`)
- Active nav link underline (if applicable)

Secondary surfaces:
- `--gray-2` (`#242428`): borders, HR dividers, sidebar border-right, breadcrumb bar border
- `--gray-3` (`#4C4C50`): inline code background, unfocused text
- `--gray-4` (`#828286`): body text (P, Span, LI)
- `--gray-5` (`#959597`): body paragraph text, navbar link color, tagline color
- `--gray-9` (`#F0F0F4`): headings, button text on brand background

Dark-only: no light mode toggle for v1. `color-scheme: dark` set globally.

Source: `src/styles/globals.css`

## Page-Level Layout Contracts

### Homepage (`/`)

- No navbar, no footer (`NO_CHROME_PATHS = ["/"]`)
- Full-viewport centered flex column: `min-height: 100dvh`, `align-items: center`, `justify-content: center`
- Vertical padding: 32px top and bottom
- Stack order: `AnimatedTerminal` ‚Üí tagline `P` ‚Üí CTA button row
- Terminal wrapper: `margin-bottom: 20px` ‚Äî see Pixel-Perfect Ghostty Constants
- Tagline: `margin-bottom: 20px` ‚Äî see Pixel-Perfect Ghostty Constants; `text-align: center`, color `--gray-5`
- CTA row: `display: flex`, `gap: 20px` ‚Äî see Pixel-Perfect Ghostty Constants; `justify-content: center`, `flex-wrap: wrap`
- Primary button: `brand` theme, `large` size
- Secondary button: `neutral` theme, `large` size

### Docs page (`/docs`, `/docs/*`)

- Navbar + footer present
- Three-column layout: sidebar (320px) | content (max 800px) | sidecar (235px, margin 24px) ‚Äî see Pixel-Perfect Ghostty Constants
- Sidebar: sticky, `top: var(--header-height)`, hidden on `max-width: 784px`
- Sidebar padding: 24px 14px
- Breadcrumb bar: `padding: 12px 0 30px` ‚Äî see Pixel-Perfect Ghostty Constants; border-bottom `--gray-2`
- Page heading block: `padding: 30px 0` ‚Äî see Pixel-Perfect Ghostty Constants; border-bottom `--gray-2`
- Description under heading: `margin-top: 8px`, color `--gray-5`, `max-width: 70%`
- `ScrollToTopButton`: fixed, `bottom: 20px` ‚Äî see Pixel-Perfect Ghostty Constants; `right: 20px` ‚Äî see Pixel-Perfect Ghostty Constants; 40√ó40px, background `--gray-2`
- `hideSidecar: true` on `docs/index.mdx` ‚Äî sidecar hidden on landing page

### Navbar (desktop)

- Link gap: `gap: 30px` ‚Äî see Pixel-Perfect Ghostty Constants

### 404 page (`/not-found`)

- Navbar + footer present (not in `NO_CHROME_PATHS`)
- Mirror Ghostty `not-found.tsx` layout ‚Äî adapt copy only

Source: Ghostty `home-content.module.css`, Ghostty `DocsPage.module.css`, `03-CONTEXT.md`

## Interaction Contracts

| Interaction | Behavior |
|-------------|----------|
| Navbar hamburger (mobile) | Toggles full-screen nav overlay; touch target = full header height (72px); `body.noScroll` applied when open |
| Navbar desktop links | Color `--gray-5` at rest; `--gray-9` on hover; underline when active route |
| CTA button hover | `brand`: opacity 70% via `hsl(var(--brand-color-hsl)/.7)`; `neutral`: background `--gray-9` |
| CTA button active | Same as hover state |
| Sidecar TOC | Active heading tracked via `useDocsStore`; auto-scrolls to active item |
| JumplinkHeader | Deeplink copy button appears on hover; updates `useDocsStore` on enter/leave viewport |
| CodeBlock | Copy-to-clipboard button; copies raw text content |
| ScrollToTopButton | Fixed position; scrolls to top on click; transition `background .2s` |
| AnimatedTerminal | `requestAnimationFrame` loop through 235 frames; Konami code easter egg |
| Text selection | Highlight color: `color-mix(in srgb, var(--brand-color) 60%, transparent)` |
