## Interaction Contracts

### Link / ButtonLink
- `Link`: underline on hover; external links show lucide `ExternalLink` icon (16px) with arrow paths translating `(2px, -2px)` on hover
- `ButtonLink brand`: border `#3A5ECF` at rest; background fills `#3A5ECF` on hover; transition `background-color 0.15s`
- `ButtonLink neutral`: border `--gray-3` at rest; background fills `--gray-3` on hover
- Border radius: 6px on all ButtonLink variants

### Navbar
- Sticky top, `z-index: 1`, height `72px` (`--header-height`)
- Desktop (>768px): logo left, nav links center-right, CTA button right
- Mobile (≤768px): logo left, hamburger button right (full header height hit area, `padding: 0 24px`)
- Hamburger open: layers animate to X (100ms ease-in-out); `prefers-reduced-motion` disables transition
- Mobile nav panel: `height: calc(100dvh - var(--header-height))`, `overflow: scroll`, `padding: 24px`
- Active nav link: `text-decoration: underline`
- Nav link hover: color transitions to `--gray-9`

### NavTree
- Folder nodes: collapsible, toggle on click
- Link nodes: standard `Link` component
- Break nodes: visual separator

### Sidecar
- Tracks active heading via `useDocsStore` (`headerIdsInView`)
- Auto-scrolls to active item
- `JumplinkHeader` must be in DOM for store to populate — implement both in same plan (Plan 02-02)

### JumplinkHeader
- Deeplink copy button appears on hover
- `useInView` integration updates `useDocsStore`

### CodeBlock
- Copy button: `opacity: 0` at rest, `opacity: 1` on parent hover or button focus
- Copy success: icon color changes to `--atom-one-green`
- Transition: `background-color 0.25s ease-in-out, opacity 0.25s ease-in-out`
- Copy button position: `absolute`, `right: 12px`, `top: 12px`

### ScrollToTopButton
- Fixed position
- Visible only when scrolled past threshold (copy Ghostty logic verbatim)

### AnimatedTerminal
- `requestAnimationFrame` loop for frame playback
- Konami code (↑↑↓↓←→←→BA) bumps FPS to 60 — keep verbatim (CONTEXT.md D-06)
