Create scripts/generate-eye-frames.js as a standalone Node.js script (no external deps, 'use strict', CommonJS require). The script must implement these exact functions and behaviors:

**Constants:**
```javascript
const FRAMES_DIR = path.join(__dirname, '../terminals/home/animation_frames');
const BACKUP_DIR = path.join(__dirname, '../terminals/home/animation_frames_backup');
const STAGE_SHAPES = {
  1: null,                              // slit — repeat '_' to fill width
  2: ['(', ' ', '_', ' ', ')'],         // half-open
  3: ['(', ' ', 'o', ' ', ')'],         // three-quarter
  4: ['(', ' ', 'O', ' ', ')'],         // full-round
};
```

**getStage(frameNum):** Returns 0-4 per D-02:
- frameNum <= 40 → 0 (squint, no change)
- frameNum <= 80 → 1 (slit)
- frameNum <= 130 → 2 (half-open)
- frameNum <= 180 → 3 (three-quarter)
- else → 4 (full-round)

**ensureBackup():** If BACKUP_DIR does not exist, create it with fs.mkdirSync({recursive:true}), then copy all .txt files from FRAMES_DIR to BACKUP_DIR with fs.copyFileSync. Log count. If BACKUP_DIR already exists, skip silently (idempotent).

**findEyeRows(lines):** Iterate all lines, collect indices where line includes '<span class="e">'. Assert exactly 10 found (throw if not). Return array of 10 indices.

**splitIntoClusters(line):** Use regex /<span class="e">[^<]+<\/span>/g to find all e-spans. Group consecutive spans (no gap between them) into clusters. A gap is any non-empty string between end of one span and start of next. Return array of cluster objects: {start: charIndex, end: charIndex, spanCount: N}. Rows 0-2 and 6-9 return 1 cluster; rows 3-5 return 2 clusters.

**buildReplacement(width, stage):**
- stage 1: return '_'.repeat(width) wrapped as individual spans: Array(width).fill('_').map(c => `<span class="e">${c}</span>`).join('')
- stages 2-4: shape = STAGE_SHAPES[stage]. If shape.length === width: use as-is. If shape.length < width: center-pad with spaces (leftPad = Math.floor((width-shape.length)/2), rightPad = remainder). If shape.length > width: trim symmetrically (trimLeft = Math.floor((shape.length-width)/2), take width chars). Map each char: '>' → '&gt;', '<' → '&lt;', '&' → '&amp;', others as-is. Wrap each in `<span class="e">${encoded}</span>`. Join with ''.

**processFrame(filePath, frameNum):**
1. Read file with fs.readFileSync(filePath, 'utf8'), split by '\n'
2. Store originalLines = [...lines]
3. stage = getStage(frameNum)
4. If stage === 0: return (no change)
5. eyeRowIndices = findEyeRows(lines)
6. For each eyeRowIndex in eyeRowIndices:
   - clusters = splitIntoClusters(lines[eyeRowIndex])
   - For each cluster: replacement = buildReplacement(cluster.spanCount, stage)
   - Replace the substring lines[eyeRowIndex].slice(cluster.start, cluster.end) with replacement
   - (Replace from right to left if multiple clusters to preserve indices)
7. Write modified lines back with fs.writeFileSync(filePath, lines.join('\n'), 'utf8')
8. Return {originalLines, modifiedLines: lines}

**validateFrame(originalLines, modifiedLines, frameFile):**
Collect errors array. Check:
1. originalLines.length === modifiedLines.length (line count unchanged)
2. For each line i: visible length unchanged. Compute visible length as: line.replace(/<[^>]+>/g, '').replace(/&gt;/g, '>').replace(/&lt;/g, '<').replace(/&amp;/g, '&').length
3. For each modified line: count '<span' occurrences === count '</span>' occurrences (all spans closed)
4. For each modified line: no raw '>' or '<' outside of tag syntax. Check: strip all <tag> patterns, then assert no '<' or '>' remain. Use: line.replace(/<[^>]+>/g, '').replace(/&gt;/g, '').replace(/&lt;/g, '').replace(/&amp;/g, '') — result should contain no '<' or '>'
5. For each modified line: all spans containing eye chars have class="e". Check: /<span(?! class="e")[^>]*>[^<]*<\/span>/ should not match on eye rows (lines that had class="e" spans originally)
Return errors array (empty = pass).

**main():**
1. ensureBackup()
2. files = fs.readdirSync(FRAMES_DIR).filter(f => f.endsWith('.txt')).sort()
3. Assert files.length === 235 (warn if not, don't throw)
4. allErrors = []
5. For each file (sorted):
   - frameNum = parseInt(file.match(/frame_(\d+)\.txt/)[1], 10)
   - result = processFrame(path.join(FRAMES_DIR, file), frameNum)
   - if result (stage > 0): errors = validateFrame(result.originalLines, result.modifiedLines, file); if errors.length > 0: allErrors.push({file, errors})
6. If allErrors.length > 0: console.error all errors, process.exit(1)
7. console.log('Done. All frames validated successfully.')

Call main() at bottom of file.
