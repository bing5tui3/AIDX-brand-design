# Plan: Brand Asset Consolidation

## Goal
Move all scattered brand files from the repo root into proper project directories (`src/assets/brand/`, `public/`, `scripts/`), fix the one broken import in the navbar, and verify the static build still passes.

## Context
- All source files are currently untracked in the repo root.
- The navbar at `src/components/navbar/index.tsx` imports `AidxWordmark` from `"./aidx-wordmark.svg"` (a local copy). After this plan, the canonical SVG lives at `src/assets/brand/aidx-wordmark.svg` and the navbar imports from `"@/assets/brand/aidx-wordmark.svg"`. The local copy (`src/components/navbar/aidx-wordmark.svg`) is then deleted.
- `public/` files are served verbatim by Next.js at the root URL path, so `public/brand-report.html` is reachable at `http://localhost:3000/brand-report.html`.
- The `scripts/` directory does not yet exist; it must be created.
