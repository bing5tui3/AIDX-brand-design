# Requirements: AIDX Website

**Defined:** 2026-04-16
**Core Value:** A polished, fully-static Next.js site with animated terminal hero and MDX docs — mirroring Ghostty website architecture exactly.

## v1 Requirements

### Project Setup

- [ ] **SETUP-01**: Project has Next.js 15+, TypeScript, Biome configured with same settings as Ghostty website
- [ ] **SETUP-02**: `package.json` has all required dependencies (next, react, @next/mdx, gray-matter, remark-*, rehype-highlight, classnames, lucide-react, zustand, slugify, fast-xml-parser)
- [ ] **SETUP-03**: `tsconfig.json` configured with `@/*` path alias pointing to `./src/*`
- [ ] **SETUP-04**: `biome.json` configured for TypeScript/TSX linting and formatting
- [ ] **SETUP-05**: `next.config.mjs` configured with MDX pipeline (remark-frontmatter, remark-gfm, custom heading IDs plugin, rehype-highlight plugin)

### Design Tokens & Global Styles

- [ ] **STYLE-01**: `src/styles/globals.css` defines all CSS custom properties: `--brand-color: #3A5ECF`, gray scale (`--gray-0` through `--gray-9`), `--header-height`, `--footer-height`, callout colors, atom-one syntax highlight colors
- [ ] **STYLE-02**: Print mode inverts color scheme via `@media print`
- [ ] **STYLE-03**: Syntax highlighting classes (`.hljs-*`) defined globally

### Fonts

- [ ] **FONT-01**: Pretendard Std Variable loaded as local font from `src/components/text/font/pretendard-std-variable.woff2`
- [ ] **FONT-02**: JetBrains Mono loaded from Google Fonts

### Core Components

- [ ] **COMP-01**: `Text` component with `as`, `font`, `weight` props; exports `H1`–`H6`, `P`, `Span`, `Code`, `LI`, `BodyParagraph`
- [ ] **COMP-02**: `Link` component with external icon detection; `ButtonLink` with `size` and `theme` variants
- [ ] **COMP-03**: `GridContainer` with `StandardGridConfig` and `NavAndFooterGridConfig`
- [ ] **COMP-04**: `Terminal` component with macOS/Adwaita platform styles, font size variants (`xtiny`/`tiny`/`small`/`medium`/`large`), columns/rows sizing
- [ ] **COMP-05**: `AnimatedTerminal` wrapping `Terminal` with `requestAnimationFrame` loop, Konami code easter egg
- [ ] **COMP-06**: `Navbar` with logo, desktop nav links, CTA button, mobile hamburger menu with full-screen nav tree
- [ ] **COMP-07**: `Footer` with nav links and copyright
- [ ] **COMP-08**: `NavTree` with `FolderNode`, `LinkNode`, `BreakNode` types; collapsible folders
- [ ] **COMP-09**: `Sidecar` with active header tracking via `useDocsStore` (zustand), auto-scroll to active item
- [ ] **COMP-10**: `Breadcrumbs` component
- [ ] **COMP-11**: `JumplinkHeader` with deeplink copy button, `useInView` integration, `useDocsStore` updates
- [ ] **COMP-12**: `CodeBlock` with copy-to-clipboard button
- [ ] **COMP-13**: `Callout` with `note`/`tip`/`important`/`warning`/`caution` types; named exports `Note`, `Tip`, `Important`, `Warning`, `Caution`
- [ ] **COMP-14**: `Blockquote` component
- [ ] **COMP-15**: `PathnameFilter` for conditional rendering by route
- [ ] **COMP-16**: `SectionWrapper` with responsive padding
- [ ] **COMP-17**: `ScrollToTopButton` fixed-position button
- [ ] **COMP-18**: `PreviewBanner` (renders null unless `GIT_COMMIT_REF=tip`)

### Lib / Utilities

- [ ] **LIB-01**: `src/lib/docs/config.ts` with `DOCS_DIRECTORY`, `DOCS_PAGES_ROOT_PATH`, `GITHUB_REPO_URL` constants
- [ ] **LIB-02**: `src/lib/docs/navigation.ts` with `loadDocsNavTreeData`, `navTreeToBreadcrumbs`, `docsMetadataTitle`
- [ ] **LIB-03**: `src/lib/docs/page.ts` with `loadDocsPage`, `loadAllDocsPageSlugs`, `DocsPageData` type
- [ ] **LIB-04**: `src/lib/docs/store.ts` zustand store for `headerIdsInView`
- [ ] **LIB-05**: `src/lib/docs/remark-heading-ids.mjs` plugin for stable heading IDs
- [ ] **LIB-06**: `src/lib/docs/remark-gfm-alerts-as-callouts.mjs` plugin mapping GFM alerts to `<Callout>`
- [ ] **LIB-07**: `src/lib/docs/rehype-highlight-all.mjs` plugin enabling full lowlight language set
- [ ] **LIB-08**: `src/lib/docs/docs-mdx.module.css` styles for MDX content (lists, images, hr, footnotes, etc.)
- [ ] **LIB-09**: `src/types/style.ts` with `Unit`, `UnitProp`, `SpacingProp` types

### Pages & Routing

- [ ] **PAGE-01**: `src/app/layout.tsx` — root layout with fonts, navbar (excluded on `/`), footer (excluded on `/`), metadata, docs nav tree loaded once
- [ ] **PAGE-02**: `src/app/page.tsx` — homepage loads terminal animation frames, renders `HomeContent`
- [ ] **PAGE-03**: `src/app/HomeContent.tsx` — client component with `AnimatedTerminal`, tagline, CTA buttons (responsive font sizing)
- [ ] **PAGE-04**: `src/app/terminal-data.tsx` — `loadAllTerminalFiles` server utility reading `./terminals/` directory
- [ ] **PAGE-05**: `src/app/docs/[[...path]]/page.tsx` — catch-all docs route with `generateStaticParams`, loads nav tree + page data + breadcrumbs
- [ ] **PAGE-06**: `src/app/docs/DocsPageContent.tsx` — docs layout with sidebar nav, content area, sidecar TOC, edit-on-GitHub link
- [ ] **PAGE-07**: `src/app/not-found.tsx` — 404 page
- [ ] **PAGE-08**: `mdx-components.tsx` — MDX component map (headings → JumplinkHeader, p → BodyParagraph, pre → CodeBlock, blockquote → Blockquote, Callout variants, CardLinks, ButtonLinks, Video)

### Docs Content

- [ ] **DOCS-01**: `docs/nav.json` with at least one section and two pages
- [ ] **DOCS-02**: `docs/index.mdx` — docs landing page with frontmatter `title` and `description`
- [ ] **DOCS-03**: At least one additional MDX doc page demonstrating headings, code blocks, callouts

### Public Assets

- [ ] **ASSET-01**: `public/` contains favicon files (favicon.ico, favicon-16.png, favicon-32.png, favicon-128.png, favicon-256.png)
- [ ] **ASSET-02**: `public/` contains AIDX logo SVG (`aidx-logo.svg`)
- [ ] **ASSET-03**: Navbar SVG wordmark at `src/components/navbar/aidx-wordmark.svg`

### Build & Quality

- [ ] **BUILD-01**: `npm run build` completes without errors
- [ ] **BUILD-02**: `npm run lint` passes with no errors or warnings
- [ ] **BUILD-03**: Site is fully static (no server-side rendering, no API routes)
